# 🚀 Arvanex Ultra Hub V3
**The Ultimate PC Games Launcher & Manager**

Arvanex Ultra Hub is a high-performance, modern gaming dashboard built with Python. It centralizes your gaming library by scanning multiple launchers and drives, providing a sleek interface with real-time Discord integration.

---

## ✨ Key Features

* 🔍 **Smart Multi-Drive Scanning:** Intelligent search across all local drives (C:, D:, E:, etc.) to locate installed games from Steam, Epic Games, Riot, and more.
* 🖼️ **Auto Icon Extraction:** Automatically extracts high-quality icons directly from game executables for a premium visual experience.
* 🎮 **Discord Rich Presence (RPC):** Show off your status! Your Discord profile will automatically display the game you're currently playing through Arvanex.
* 👤 **Active User Detection:** Seamlessly identifies the active Steam user and displays the profile name in the sidebar.
* ⚡ **Database Optimized:** Saves your library to a local JSON database for near-instant startup after the initial scan.
* 🎨 **Modern UI:** Built with a dark-themed, responsive interface using CustomTkinter.

🚀 How to Use
Download: Clone the repository or download the latest release .exe.

Initial Scan: On the first launch, Arvanex will perform a "Fancy Scan" to build your library. This happens only once.

Launch: Click "LAUNCH" on any game to start playing.

Discord: Make sure Discord is open to see your "Playing Status" update automatically.
